스타벅스 APP 개선 디자인

STARBUCKS APP Redesign

Mobile

## plan
기존 스타벅스앱은 다양한 기능을 제공하고 있음에도 불구하고, 직관적이지 않은 구조와 설명 부족으로 사용자가 기능을 인지하고 활용하기 어려웠습니다. 또한 사용자는 단순한 주문 기능을 넘어, 브랜드 감성과 개인화된 ‘특별한 경험’을 기대하고 있었습니다. 이러한 인사이트를 바탕으로 시크릿 메뉴 추천 기능을 핵심으로 설정하고, 기능 가이드 강화와 주문 상태 피드백, 지갑 기능 개선을 통해 전반적인 사용 흐름을 함께 개선했습니다.

Mobile

## plan
